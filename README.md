# Next.js App

This project was bootstrapped with Next.js using TypeScript, ESLint, App Router, and npm.

## Getting Started

### Development

```
npm run dev
```

### Build

```
npm run build
```

### Start (Production)

```
npm start
```

## Features

- TypeScript
- ESLint
- App Router
- src directory structure
